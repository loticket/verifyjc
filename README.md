###  投注格式解析

例如：190225-001(10,20,21);190225-002(31,32,21)  2串1
190225-001(10,20,21) 玩法id:赛程日期-场次(投注内容)
190225:当前赛程日历
002:场次
(10,20,21):投注信息

混合过关实例：40:190225-001(3);40:190225-002(3)  2串1
40:190225-001(3) 玩法id:赛程日期-场次(投注内容)
